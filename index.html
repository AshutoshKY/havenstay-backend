Welcome to the HavenStay Backend API documentation! 🚀

HavenStay is a modern, high-performance Domain-Driven Design (DDD) backend built in Go. It empowers Hosts to manage luxury properties and Guests to seamlessly book remote getaways.

Key Features:

  • 🏘️ Property Management: Full CRUD lifecycle for short-term rentals and luxury homes.
  • 👩‍💼 Host & Guest Profiles: Independent onboarding and profile management domains.
  • 📅 Booking Engine: Real-time reservation creation.
  • ❤️ Wishlists: Guests can curate lists of favorite properties.

Built With: Go 1.21+, Gin (REST), gRPC, Protobufs, MySQL 8, GORM.
